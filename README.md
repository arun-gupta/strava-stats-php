# Strava Activity Analyzer (PHP)

A PHP application for analyzing and visualizing Strava activity statistics.

## Installation

```bash
composer install
```

## Configuration

Copy `.env.example` to `.env` and configure your Strava API credentials:

```bash
cp .env.example .env
```
